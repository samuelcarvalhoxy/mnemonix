Como usar o Mnemônix 🧠

🔢 O Sistema Maior

Cada dígito (0-9) é associado a sons consonantais (ex: 1=T/D, 2=N, 3=M...). Juntando dígitos, criamos palavras concretas preenchendo com vogais (ex: 12 = ToNnel).

📖 Dicionário Completo

Consulte a base de 110 itens (0-9 e 00-99). Toque em um item para criar sua própria palavra ou adicionar uma imagem mental marcante.

🎴 Flashcards

Treine a associação número ↔ palavra. Marque "Fácil" para subir o domínio da carta, ou "Difícil" para refazê-la no mesmo treino. O modo reverso inverte a lógica.

⚡ Sequências Numéricas

O teste final de memorização. O app gera números aleatórios. Você cria uma história mental unindo as palavras. O Modo Velocidade esconde os números automaticamente após o tempo acabar.

⚙️ Backup Seguro

Todo seu progresso e customizações ficam salvos apenas no seu celular (offline). Use as Configurações para exportar um backup JSON seguro.
